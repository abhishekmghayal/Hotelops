# 🏨 HotelOps

HotelOps is a smart hotel operations management system that replaces WhatsApp messages and handwritten checklists with a real-time digital platform.

It helps manage:
- Housekeeping tasks
- Maintenance requests
- Front desk operations
- Room status tracking
- Manager dashboard & analytics

---

## 🚀 Features

- 🔐 Authentication (JWT-based login)
- 🏨 Room management (Ready, Dirty, Occupied, etc.)
- 📋 Task management (Cleaning, Maintenance, Guest requests)
- 🧹 Housekeeping workflow with checklist
- 🔧 Maintenance ticket system
- 📊 Dashboard with real-time stats
- 🔔 Notifications system
- ⚡ Real-time updates using Socket.io

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Recharts

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Socket.io
- Multer (file upload)
